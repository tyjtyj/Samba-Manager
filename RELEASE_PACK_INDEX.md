### Build Script Fails
```bash
# Check Python
python3 --version  # Need 3.6+

# Check permissions
ls -la build_release.sh  # Should have x permission

# Clear and retry
rm -rf build/
./build_release.sh
```

### Checksum Mismatch
```bash

cd releases/stable/
sha256sum *.tar.gz *.zip > checksums.txt

# Verify
sha256sum -c checksums.txt
```

```bash

sudo apt install gh  # Debian

# Authenticate
```
